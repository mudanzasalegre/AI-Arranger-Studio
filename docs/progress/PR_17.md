# PR-17 - Epic 27 Tokenization Dataset Export

Fecha: 2026-06-29

## Alcance implementado

- Nuevo paquete `packages/training` para preparar datasets tokenizados sin entrenar modelos.
- Exportador reproducible desde `dataset_tools.PatternIndex`:
  - roles: `melody`, `bass`, `piano_comping`, `horn_responses`, `drums`;
  - splits deterministas `train/val/test`;
  - filtrado por `usable_for_training`, calidad minima y licencias bloqueadas;
  - metadata por segmento con licencia, source hash, fingerprint, chord context,
    section context, tags, estilo y sensibilidad comercial.
- Tokenizer simbolico `MidiTokBridgeTokenizer` con config `miditok_config.json`,
  sin importar MidiTok ni dependencias pesadas en runtime.
- CLI reproducible:

```powershell
python scripts/export_tokenized_dataset.py --pattern-index path\to\pattern_index.json --output-dir outputs\tokenized_dataset
```

- Smoke test especifico:

```powershell
python scripts/tokenization_dataset_smoke.py
```

- `requirements.txt`, `pyproject.toml`, `scripts/bootstrap_check.py` y `Makefile`
  actualizados para reconocer el nuevo paquete `packages/training`.
