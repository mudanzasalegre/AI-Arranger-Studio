# PR-19 - Epic 29 Custom Role Model Interface

Fecha: 2026-06-29

## Alcance implementado

- Interfaces de generador por rol:
  - `AIWalkingBassGenerator`;
  - `AIPianoCompingGenerator`;
  - `AIMelodyGenerator`;
  - `AIHornResponseGenerator`;
  - `AIDrumsGenerator`.
- Todos usan el mismo contrato:
  - `ModelBackend.generate(ModelRequest)`;
  - salida simbolica `NOTE|...`;
  - conversion a `ArrangementProject.Track` sin tocar exportadores ni API principal.
- Modos soportados por metadata:
  - `rule_based`;
  - `retrieval`;
  - `external_model`;
  - `custom_model`.
- `RuleBasedArranger` ahora permite inyectar generadores de drums, bass, piano,
  melody y horn responses desde constructor, manteniendo defaults rule-based.
- `DeterministicRoleModelBackend` cubre todos los roles para tests/smoke sin
  dependencias pesadas.
- `.gitignore` ajustado de `models/` a `/models/` para seguir ignorando modelos
  pesados de raiz sin excluir codigo fuente como `packages/training/training/models`.
