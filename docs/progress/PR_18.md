# PR-18 - Epic 28 Baseline Statistical Role Models

Fecha: 2026-06-29

## Alcance implementado

- Nueva capa `training.models.statistical` para entrenar modelos estadisticos
  baseline desde `tokenized_segments.jsonl`.
- Modelos por rol:
  - `melody`: `ngram_melody`;
  - `bass`: `ngram_bass`;
  - `piano_comping`: `markov_voicings`;
  - `horn_responses`: `pattern_probability_model`;
  - `drums`: `drum_fill_retrieval`.
- Modelo global adicional `pattern_probability_model.json`.
- Cada artefacto guarda vocabulario, conteos de tokens, conteos de contexto,
  probabilidades de patron, patrones representativos, splits y metadata de
  licencia/source.
- `StatisticalRoleModel` carga artefactos, genera tokens de forma determinista y
  calcula `negative_log_likelihood`/`perplexity`.
- Reporte `baseline_comparison.json` con:
  - metricas del modelo estadistico;
  - proxy `rule_based` por vocabulario uniforme;
  - proxy `retrieval` por mejor similitud Jaccard contra train.
